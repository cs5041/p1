Asteroids [Reloaded] (adapted for CS5041)

Instructions (only for keyboard testing):

F: Force start game without micro:bit

Left and Right Arrows: Rotate ship

Up Arrow: Thrust

Space: Fire main weapon(s)

Z: Fire bomb weapon (area effect, uses energy)

Down Arrow or Shift: Shield (will recharge...)

S: Enable or disable sound effects

Escape: Pause the game

DEBUG Keys:

L: Skip level

A: Add asteroid

E: Add enemy ship
